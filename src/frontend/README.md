# SCOOPEX Shift - Frontend

Modern React + TypeScript frontend for SCOOPEX Shift, a comprehensive shifting and logistics platform for the Mumbai-Karjat corridor.

## Features

- 🚚 Instant price estimation for shifting services
- 📍 Google Maps integration with address autocomplete
- 💳 Razorpay payment integration
- 🔐 Internet Identity authentication
- 📱 Fully responsive design
- 🌓 Light/dark mode support
- ⚡ Built with Vite for fast development

## Prerequisites

- Node.js 18+ and pnpm
- Google Maps API key (see [GOOGLE_MAPS_SETUP.md](./GOOGLE_MAPS_SETUP.md))
- Razorpay account for payment processing
